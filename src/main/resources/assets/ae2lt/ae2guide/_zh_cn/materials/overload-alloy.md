# 过载合金

**过载合金锭**是一种高级金属材料，在许多高级配方中都会用到。

## 加工流程

1. **过载合金锭粗胚** — 未精炼的中间产物
2. **过载合金锭** — 由粗胚精炼而成的成品合金锭
3. **过载合金板** — 由过载合金锭压制而成，用于多种终极配方

## 物品一览

| 物品 | 作用 |
|------|------|
| 过载合金锭粗胚 | 中间产物 |
| 过载合金锭 | 成品合金锭，关键合成材料 |
| 过载合金板 | 压制板材，用于极限过载核心等高级配方 |
