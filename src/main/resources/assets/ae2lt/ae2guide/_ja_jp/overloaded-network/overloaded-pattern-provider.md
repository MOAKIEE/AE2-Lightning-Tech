# オーバーロードパターンプロバイダー

<Row>
  <BlockImage id="ae2lt:overloaded_pattern_provider" scale="4" />
</Row>

**オーバーロードパターンプロバイダー**はバニラ <ItemLink id="ae2:pattern_provider" /> の拡張版で、**36 個のパターンスロット**を持ち、**無線モード**をサポートし、距離をまたいで遠隔マシンに加工素材を派遣できます。

## 主要特性

* **36 個のパターンスロット**（バニラは 9 個）
* **2 つの稼働モード**：通常モードと無線モード
* **オーバーロードパターン対応**：バニラパターンに加え、本 MOD のオーバーロードパターンもサポート
* **自動回収**：リモートマシンから加工産物を自動的に取り戻し
* **入力フィルタ**：パターンレシピが定義した産物のみ ME ネットワークに戻ることを許可

## 稼働モード

### 通常モード

通常モードでは、オーバーロードパターンプロバイダーは素材を**物理的に隣接する**マシンに送り込み、挙動はバニラのパターンプロバイダーと一致します。主な相違はより多くのパターンスロットを持つことです。

### 無線モード

<ItemImage id="ae2lt:overloaded_wireless_connect_tool" scale="2" float="left" />

無線モードでは、オーバーロードパターンプロバイダーは距離をまたいで素材をリモートマシンに派遣できます。無線接続の確立には**オーバーロード無線接続ツール**を使用します：

1. オーバーロード無線接続ツールを所持し、**Shift + 右クリック**でオーバーロードパターンプロバイダーを選択
2. 対象マシンの特定の面を**右クリック**して接続を確立
3. 同一プロバイダーに複数のリモートマシンをバインドできます

接続完了後、プロバイダーは選択された割当戦略に基づきリモートマシンに素材を派遣します。

### 割当戦略

| 戦略 | 説明 |
|------|------|
| 順次配布 | 順序に従い 1 台ずつリモートマシンに派遣 |
| 均等配布 | 接続された全リモートマシンに素材を均等に分配 |

## 回収モード

回収モードは加工完了後に産物がリモートマシンからどのように取り戻されるかを決定します：

| モード | 説明 |
|------|------|
| オフ | 自動回収しない |
| 自動 | リモートマシンから能動的に産物を抽出 |
| プッシュ | リモートマシンが仮想出力口に産物を送る；プロバイダーが受動的に受信 |

## 速度モード

| 速度段 | 説明 |
|-------|------|
| 通常 | 標準クールダウン時間（5 ~ 80 tick） |
| 高速（プローブ） | 自適応クールダウン、プローブメカニズムで準備状態を事前検出（1 ~ 40 tick） |

## 入力フィルタ

「入力フィルタ」を有効化すると、プロバイダーは産物回収時に現在のパターンで定義された出力アイテムのみを受信し、誤ったアイテムがネットワークに侵入することを防ぎます。

## オーバーロードパターンとオーバーロードパターンエンコーダー

<Row>
  <ItemImage id="ae2lt:overload_pattern" scale="2" />
  <ItemImage id="ae2lt:overload_pattern_encoder" scale="2" />
</Row>

**オーバーロードパターン**は専用パターンで、**オーバーロードパターンプロバイダー内でのみ使用可能**です。**オーバーロードパターンエンコーダー**を使用してエンコードします。

オーバーロードパターンエンコーダーは以下をサポートします：

* メイン入力とメイン出力の設定
* 副産物の設定
* **NBT 無視**スイッチ：有効化するとアイテムマッチング時に NBT データを無視

## 自動化のアドバイス

* 大規模自動合成システムでは、無線モードにより配線の複雑さを大幅に低減できます
* 「均等配布」戦略と複数の加工マシンを組み合わせることで並列加工を実現可能
* 「高速」速度段の有効化により応答性を向上することを推奨
* 自動化シーンに応じて適切な回収モードを選択
