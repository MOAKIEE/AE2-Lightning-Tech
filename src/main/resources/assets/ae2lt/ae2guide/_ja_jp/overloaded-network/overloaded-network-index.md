# オーバーロードネットワーク

AE2 Lightning Tech は「オーバーロード」ネットワーク設備のセットを提供し、大型基地でバニラ AE2 のチャンネルと容量の制限を突破するために使用されます。

バニラと比較して：

* 各**オーバーロード ME コントローラー**がネットワークに 128 チャンネルを追加で提供
* ME オーバーロードケーブル自体にチャンネル上限が存在せず、オーバーロードネットワーク全体はコントローラー総数に基づきグローバルにチャンネルを割当
* オーバーロードパターンプロバイダーとオーバーロード ME インターフェースは 36 個のスロットを持ち、距離をまたぐ無線接続をサポート
