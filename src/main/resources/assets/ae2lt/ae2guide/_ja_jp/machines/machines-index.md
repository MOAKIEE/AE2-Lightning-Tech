# 加工マシン

AE2 Lightning Tech は様々な加工マシンを提供し、異なるタイプの素材合成需要に対応します。大気イオナイザーが AE エネルギーを消費する以外、その他のマシンはすべて ME ネットワークに接続しつつ外部 FE エネルギーを使用し、加工時に ME ネットワークから雷を抽出します。

<SubPages />

## 共通特性

### 自動排出

ほとんどのマシンは**自動排出**をサポートします。有効にすると、マシンは加工完了時に許可された方向の隣接コンテナに産物を送り込みます。具体的な方向はマシン GUI の「出力面設定」画面で設定可能です。

### 速度カード

クリスタル触媒装置を除き、その他の加工マシンは AE2 バニラの**速度カード**装着による加工速度向上をサポートします、最大 4 枚まで装着可能です。

### 雷代替

超高電圧の雷を必要とする一部のレシピは**マトリクス代替**メカニズムをサポートします：マシン内に**雷崩壊マトリクス**が装着されている場合、数倍量の高電圧の雷で超高電圧の雷を代替して加工を完了できます。具体的な代替比率はレシピ自体によります。
