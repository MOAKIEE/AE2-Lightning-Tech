# オーバーロード処理ファクトリー

<Row>
  <BlockImage id="ae2lt:overload_processing_factory" scale="4" />
</Row>

**オーバーロード処理ファクトリー**は本 MOD で最大規模・最高スループットの加工設備です。アイテムと液体の混合入出力をサポートし、雷崩壊マトリクスでマルチ並列加工を解放し、大規模工業化生産ラインの中核マシンです。

## スロットと容量

| スロット / コンポーネント | 容量 | 説明 |
|------------|------|------|
| アイテム入力スロット ×9 | 8,192 | 加工待ちのアイテムを入れる |
| マトリクススロット ×1 | 32 | 雷崩壊マトリクスを装着して並列能力を解放 |
| アイテム出力スロット ×1 | 8,192 | 加工産物；マシンの書き込みのみを許可 |
| 液体入力スロット ×1 | 512,000 mB | パイプ経由で必要な液体を注入 |
| 液体出力スロット ×1 | 512,000 mB | 加工で産出された液体を保管 |
| FE エネルギーバッファ | 640,000,000 FE（デフォルト） | 内蔵エネルギーバッファ |
| 速度カードスロット | — | AE2 速度カードを最大 4 枚装着可能 |

## 稼働フロー

1. アイテム入力スロットと液体入力口から原料を供給
2. マシンが利用可能なレシピをマッチング
3. レシピがマッチし、ME ネットワークの雷と内部 FE エネルギーが十分であれば、マシンは加工を開始
4. 加工過程では雷と FE エネルギーを継続的に消費
5. 産物がそれぞれアイテム出力スロットと液体出力スロットに入ります

## マルチ並列加工

<ItemImage id="ae2lt:lightning_collapse_matrix" scale="2" float="left" />

オーバーロード処理ファクトリーの並列能力は装着された**雷崩壊マトリクス**の数量により決定されます。マトリクススロットは最大 32 枚のマトリクスを収容し、各マトリクスは設定に基づき一定量の並列上限を提供します（デフォルトは各 8 並列、上限 32 × 8 = 256 回）：

* マトリクス未装着時、ファクトリーは並列加工できません
* マトリクスを多く装着するほど並列上限が線形に向上
* 実際の並列数量は同時に入力素材の数量とレシピの単回使用量にも制限されます
* 並列回数が多いほど単回加工の産出量が多くなり、同時に FE エネルギー消費も線形に増加します

## 速度カードとエネルギー消費

オーバーロード処理ファクトリーは速度カードを最大 4 枚サポートします。速度カードを 1 枚追加するごとに、単 tick で消費可能な FE 上限が顕著に向上します：

| 速度カード数 | 単 tick FE 上限（デフォルト値） |
|-----------|-------------------------|
| 0 | 400,000 FE |
| 1 | 2,000,000 FE |
| 2 | 8,000,000 FE |
| 3 | 32,000,000 FE |
| 4 | 128,000,000 FE |

## 液体入力 / 出力

液体スロットは任意の液体パイプシステム経由で入出力可能です。入力スロットはレシピが要求する液体タイプを厳格に検証します。

## 自動排出

自動排出を有効にすると、アイテム産物が許可された方向の隣接コンテナに自動的に送り込まれます。具体的な方向は GUI の「出力面設定」画面で設定します。

## エネルギー供給

オーバーロード処理ファクトリーは**外部 FE エネルギー入力**で供電され、ME ネットワークから直接 AE エネルギーを抽出することはありません。任意の FE エネルギーを出力可能なパイプ / 導線でマシン側面にエネルギーを接続してください。
