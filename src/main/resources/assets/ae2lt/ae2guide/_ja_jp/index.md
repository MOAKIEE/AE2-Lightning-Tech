# AE2 Lightning Tech

AE2 Lightning Tech は Applied Energistics 2 の拡張 MOD です。ME ネットワークに新しいストレージタイプである**雷**を導入し、雷を中心とした完全な素材チェーン、加工マシン、そしてバニラのチャンネル上限を突破する「オーバーロード」ネットワーク機器を提供します。

主な特徴：

* 自然または装置によって発生した雷を**雷エネルギー**として収集し、アイテムや液体と同様に ME ネットワークに保管できます
* 雷で動作する各種の高度な加工マシンにより、オーバーロード合金、オーバーロードプロセッサ、雷崩壊マトリクスなどの新素材を生産します
* オーバーロードコントローラー、オーバーロードケーブル、オーバーロードパターンプロバイダー、オーバーロード ME インターフェースにより、ME ネットワークのチャンネル容量と管理範囲を拡張します
