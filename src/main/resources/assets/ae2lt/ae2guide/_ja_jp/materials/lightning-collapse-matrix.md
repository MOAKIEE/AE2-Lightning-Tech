# 雷崩壊マトリクス

<ItemImage id="ae2lt:lightning_collapse_matrix" scale="2" float="left" />

**雷崩壊マトリクス**は AE2 Lightning Tech の最も核心的な終末級コンポーネントの一つです。マシンの中で消耗品としてではなく、**代替と並列のための触媒要素**として機能します——様々なマシンのマトリクススロットにマトリクスを 1 枚保持しておくだけで、そのマシンの高階稼働モードが解放されます。

## 入手方法

### 雷シミュレーションルームで合成

| 素材 | 数量 |
|------|------|
| パーフェクトエレクトロチャイムクリスタル | 1 |
| 究極オーバーロードコア | 16 |
| **超高電圧の雷** | 128 |
| エネルギー | 50,000,000 AE |

コストが高めなので、オーバーロード処理ファクトリーや雷アセンブリチャンバーの生産計画と併せて計画することを推奨します。

## マシン触媒として

雷崩壊マトリクスは加工中に**消費されません**が、常にマトリクススロットに保持しておく必要があります：

* [雷シミュレーションルーム](../machines/lightning-simulation-chamber.md) / [雷アセンブリチャンバー](../machines/lightning-assembly-chamber.md) — マトリクス装着後、元々**超高電圧の雷**を要求する一部のレシピを数倍量の**高電圧の雷**で代替可能
* [テスラコイル](../lightning/tesla-coil.md) — 超高電圧モードではスロットにマトリクスの装着が必須、さもなくば高電圧の雷から超高電圧への昇圧不可
* [クリスタル触媒装置](../machines/crystal-catalyzer.md) — クリスタルモードとダストモードの両方で有効；マトリクス装着後、並列後の単回産出量がさらに **4 倍**に向上
* [オーバーロード処理ファクトリー](../machines/overload-processing-factory.md) — マルチマトリクス並列：マトリクスを 1 枚追加するごとに 1 段階の並列度を解放（スロット最大 32 枚、デフォルトで各 8 並列を提供）

## 観測メモ

> 外部チームから極めて薄い検証レポートが提出された：マトリクスが**いかなる容器にも拘束されていない**状態では予想外の反応が出現する；反応が点火されたシーンは [オーバーロード TNT](../lightning/overload-tnt.md) のページに記録されている。

産物は未登録のセルで、<ItemLink id="ae2:drive" /> に挿せば解析可能です；内部の残留物は**一度きり読み取れる**だけで、読了後は空になります。読み取れるものについて——レポートには一文だけ添えられていた：「サンプル分布は極めて不均一で、最も稀少なものはセルではなく、まだ文字の書かれていない冊子だ」。
