# オーバーロード合金

  <ItemIcon id="ae2lt:overload_alloy_blank" />
  <ItemIcon id="ae2lt:overload_alloy" />
  <ItemIcon id="ae2lt:overload_alloy_plate" />

**オーバーロード合金**は高級金属素材で、機械フレーム、オーバーロードコントローラー、その他多数の終末級コンポーネントに広く使用されます。

## 加工フロー

1. **オーバーロード合金インゴット素材** — 作業台で直接合成
2. **オーバーロード合金インゴット** — 完成品の合金インゴット、雷シミュレーションルームで素材から精錬可能、またはオーバーロード処理ファクトリーで金属と結晶系素材から一括生産可能
3. **オーバーロード合金プレート** — 雷シミュレーションルームまたはオーバーロード処理ファクトリーでオーバーロード合金インゴットからプレス成形、究極オーバーロードコアなどのトップレシピに必要な素材

<RecipeFor id="ae2lt:overload_alloy_blank" />

オーバーロード合金インゴットは 2 種類のマシンから入手可能です：

| マシン | 入力 | 雷 | エネルギー | 産出 |
|------|------|------|------|------|
| 雷シミュレーションルーム | `1` オーバーロード合金インゴット素材 + `4` オーバーロードクリスタルダスト | `1` 高電圧の雷 | `20,000` | `1` オーバーロード合金インゴット |
| オーバーロード処理ファクトリー | `1` 銅ブロック + `1` 鉄ブロック + `1` 金ブロック + `1` アメジストブロック + `1` エンダーアイ + `1` コーラスフルーツ | `1` 高電圧の雷 | `100,000` | `27` オーバーロード合金インゴット |

## アイテム一覧

| アイテム | 役割 |
|------|------|
| オーバーロード合金インゴット素材 | 中間産物 |
| オーバーロード合金インゴット | 完成品の合金インゴット、ほとんどのオーバーロードマシンの主要合成素材 |
| オーバーロード合金プレート | プレス成形プレート、究極オーバーロードコアなどの高度なレシピに使用 |
