# 素材とアイテム

AE2 Lightning Tech は多くの新素材を導入しています。以下のページでは各中核素材とアイテムを詳しく紹介します。

<SubPages />

## 天気コンデンセート

天気コンデンセートは [大気イオナイザー](../machines/atmospheric-ionizer.md) で消費され、ワールドの天気を制御するために使用されます。雷シミュレーションルーム、雷アセンブリチャンバー、またはオーバーロード処理ファクトリーで生産可能です。

<ItemGrid>
  <ItemIcon id="ae2lt:clear_condensate" />
  <ItemIcon id="ae2lt:rain_condensate" />
  <ItemIcon id="ae2lt:thunderstorm_condensate" />
</ItemGrid>

| アイテム | 目標天気 | AE 消費 |
|------|---------|---------|
| 晴天コンデンセート | 晴れ | 500,000 AE |
| 降雨コンデンセート | 雨 | 1,000,000 AE |
| 雷雨コンデンセート | 雷雨 | 8,000,000 AE |

## ブロック

| ブロック | 説明 |
|------|------|
| オーバーロードクリスタルブロック | 装飾ブロック、オーバーロードクリスタルマザーロック変換構造を構築するための素材でもある |
| [シリコンブロック](overload-machine-frame.md) | シリコンの圧縮ストレージブロック |
| [オーバーロード機械フレーム](overload-machine-frame.md) | ほとんどのオーバーロード加工マシンに共通する基礎フレーム |
| <ItemLink id="ae2lt:overload_tnt" /> | 爆発性ブロック、起爆後に持続的な人工の雷爆撃を発生；詳細は [オーバーロード TNT](../lightning/overload-tnt.md) |
