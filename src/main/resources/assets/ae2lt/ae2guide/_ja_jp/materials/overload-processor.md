# オーバーロードプロセッサ

**オーバーロードプロセッサ**は高級プロセッサで、ほとんどのオーバーロードマシンや高度な設備を作成するために使用されます。その生産プロセスは AE2 インスクライバーを使用した複数ステップ合成が必要です。

## 合成フロー

1. **オーバーロード金型**を入手（雷変換、雷シミュレーションルーム、またはオーバーロード処理ファクトリーで入手可能）
2. AE2 インスクライバーでオーバーロード金型とオーバーロード合金インゴット素材を使用し、**未オーバーロード回路基板**をプレスする
3. 未オーバーロード回路基板をインスクライバーに戻し、**オーバーロード回路基板**に加工する
4. オーバーロード回路基板を AE2 分子組立機で他の素材と組み合わせることで、**オーバーロードプロセッサ**を入手できます

## アイテム一覧

| アイテム | 役割 |
|------|------|
| オーバーロード金型 | AE2 インスクライバーで使用する金型；雷変換またはマシン加工で入手 |
| 未オーバーロード回路基板 | 中間産物；オーバーロード金型でオーバーロード合金インゴット素材をプレスして得る |
| オーバーロード回路基板 | 完成品の回路基板；未オーバーロード回路基板をインスクライバーでさらに加工して得る |
| オーバーロードプロセッサ | 最終プロセッサ、マシンや設備の合成レシピに使用 |
