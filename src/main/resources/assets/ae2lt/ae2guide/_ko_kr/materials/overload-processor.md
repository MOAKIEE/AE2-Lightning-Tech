# 과부하 프로세서

  <ItemIcon id="ae2lt:overload_processor" />

**과부하 프로세서**는 고급 프로세서로, 대부분의 과부하 기계와 고급 설비의 제작에 사용됩니다. 그 생산 과정은 AE2 압인기를 통한 다단계 합성이 필요합니다.

## 합성 흐름

1. **과부하 회로 프레스**를 획득(번개 변환, 번개 시뮬레이션 룸 또는 과부하 처리 공장에서 획득)
2. AE2 압인기에서 과부하 회로 프레스와 과부하 합금괴 소재를 사용하여 **미과부하 회로 기판**을 압인
3. 미과부하 회로 기판을 압인기에 다시 넣어 **과부하 회로 기판**으로 가공
4. 과부하 회로 기판과 기타 재료를 AE2 분자 조립기에서 조합하여 **과부하 프로세서**를 획득

## 아이템 목록

| 아이템 | 용도 |
|------|------|
| 과부하 회로 프레스 | AE2 압인기에서 사용하는 압인 프레스; 번개 변환 또는 기계 가공으로 획득 |
| 미과부하 회로 기판 | 중간 산물; 과부하 회로 프레스로 과부하 합금괴 소재를 압인하여 획득 |
| 과부하 회로 기판 | 완성된 회로 기판; 미과부하 회로 기판을 압인기에서 추가 가공하여 획득 |
| 과부하 프로세서 | 최종 프로세서, 기계와 설비의 합성 레시피에 사용 |
