# 과부하 합금

  <ItemIcon id="ae2lt:overload_alloy_blank" />
  <ItemIcon id="ae2lt:overload_alloy" />
  <ItemIcon id="ae2lt:overload_alloy_plate" />

**과부하 합금**은 고급 금속 재료로, 기계 프레임, 과부하 제어기 및 다양한 종말기 구성 요소에 광범위하게 사용됩니다.

## 가공 흐름

1. **과부하 합금괴 소재** — 작업대에서 직접 합성
2. **과부하 합금괴** — 완성된 합금괴, 번개 시뮬레이션 룸에서 소재로부터 정제하거나, 과부하 처리 공장에서 금속과 결정류 재료로 일괄 생산
3. **과부하 합금판** — 번개 시뮬레이션 룸 또는 과부하 처리 공장에서 과부하 합금괴를 압연하여 만들며, 궁극 과부하 코어 등 최상위 레시피에 필요한 재료

<RecipeFor id="ae2lt:overload_alloy_blank" />

과부하 합금괴는 두 가지 기계로 획득할 수 있습니다:

| 기계 | 입력 | 번개 | 에너지 | 산출 |
|------|------|------|------|------|
| 번개 시뮬레이션 룸 | `1` 과부하 합금괴 소재 + `4` 과부하 수정 가루 | `1` 고전압 번개 | `20,000` | `1` 과부하 합금괴 |
| 과부하 처리 공장 | `1` 구리 블록 + `1` 철 블록 + `1` 금 블록 + `1` 자수정 블록 + `1` 엔더의 눈 + `1` 콜러스 열매 | `1` 고전압 번개 | `100,000` | `27` 과부하 합금괴 |

## 아이템 목록

| 아이템 | 용도 |
|------|------|
| 과부하 합금괴 소재 | 중간 산물 |
| 과부하 합금괴 | 완성된 합금괴, 대부분의 과부하 기계의 핵심 합성 재료 |
| 과부하 합금판 | 압연 판재, 궁극 과부하 코어 등 고급 레시피에 사용 |
