# 번개 붕괴 매트릭스

<ItemImage id="ae2lt:lightning_collapse_matrix" scale="2" float="left" />

**번개 붕괴 매트릭스**는 AE2 Lightning Tech에서 가장 핵심적인 종말기 구성 요소 중 하나입니다. 기계 안에서는 소모품이 아니라 **대체와 병렬의 촉매 요소**로 작용합니다 — 여러 기계의 매트릭스 슬롯에 매트릭스 한 개를 두기만 하면, 해당 기계의 고급 운용 모드가 해금됩니다.

## 획득 방법

### 번개 시뮬레이션 룸에서 합성

| 재료 | 수량 |
|------|------|
| 완벽한 전명 수정 | 1 |
| 궁극 과부하 코어 | 16 |
| **초고전압 번개** | 128 |
| 에너지 | 50,000,000 AE |

비용이 비교적 높으므로, 과부하 처리 공장 및 번개 어셈블리 챔버의 생산 일정과 함께 계획하는 것을 권장합니다.

## 기계 촉매제로서의 역할

번개 붕괴 매트릭스는 가공 중 **소비되지 않지만**, 매트릭스 슬롯에 항상 남아 있어야 합니다:

* [번개 시뮬레이션 룸](../machines/lightning-simulation-chamber.md) / [번개 어셈블리 챔버](../machines/lightning-assembly-chamber.md) — 매트릭스 장착 후, 일부 원래 **초고전압 번개**를 요구하는 레시피는 몇 배 수량의 **고전압 번개**로 대체 가능
* [테슬라 코일](../lightning/tesla-coil.md) — 초고전압 모드는 반드시 슬롯에 매트릭스를 장착해야 하며, 그렇지 않으면 고전압 번개에서 초고전압으로 승압할 수 없음
* [수정 촉매기](../machines/crystal-catalyzer.md) — 수정 모드와 가루화 모드 모두 적용; 매트릭스 장착 후 병렬화된 단회 산출이 **4배**로 추가 향상
* [과부하 처리 공장](../machines/overload-processing-factory.md) — 다중 매트릭스 병렬: 매트릭스 한 개가 추가될 때마다 병렬도 한 단계 해금(슬롯 최대 32개, 기본값 각 8 병렬 제공)

## 관찰 메모

> 외부 팀에서 매우 얇은 재검토 보고서가 제출되었다: 매트릭스가 **어떤 컨테이너에도 제약받지 않은** 상태일 때 예상치 못한 반응이 나타난다; 반응이 점화된 장면은 [과부하 TNT](../lightning/overload-tnt.md) 페이지에 기록되어 있다.

산물은 등록되지 않은 셀 한 개이며, <ItemLink id="ae2:drive" />에 연결하면 해석할 수 있다; 내부 잔류물은 **한 번만 읽을 수 있고**, 읽고 나면 비게 된다. 무엇이 읽힐지에 대해서는 — 보고서에는 한 줄만 덧붙여 있다: "샘플 분포가 극히 불균일하다, 가장 희귀한 것은 셀이 아니라, 아직 글이 쓰이지 않은 책 한 권이다."
