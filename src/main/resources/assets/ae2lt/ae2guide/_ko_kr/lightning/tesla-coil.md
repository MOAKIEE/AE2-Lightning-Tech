# 테슬라 코일

<Row>
  <BlockImage id="ae2lt:tesla_coil" scale="4" />
</Row>

**테슬라 코일**은 제어 가능하고 안정적으로 번개를 대량 생산하는 핵심 설비입니다. 날씨에 의존하지 않고, FE 에너지와 재료를 소비하여 능동적으로 번개를 산출합니다.

## 두 가지 작동 모드

테슬라 코일은 GUI에서 두 가지 모드를 전환할 수 있습니다.

### 고전압 모드

* **입력**: 과부하 수정 가루 + FE 에너지
* **출력**: 고전압 번개(ME 네트워크에 주입)
* **기본 비용**: 고전압 번개 1개 산출당 과부하 수정 가루 2개 + 25,000 FE 소비

고전압 모드는 배치 처리를 지원합니다. 코일은 현재 수정 가루 슬롯의 가용 재료 전체와 ME 네트워크가 수용 가능한 고전압 번개 상한을 고정한 후, 한 번의 충전 주기 내에서 모든 번개를 일괄 산출합니다.

### 초고전압 모드

* **입력**: 고전압 번개(ME 네트워크에서 추출) + FE 에너지, 동시에 **번개 붕괴 매트릭스** 장착 필수
* **출력**: 초고전압 번개(ME 네트워크에 주입)
* **기본 비용**: 초고전압 번개 1개 산출당 고전압 번개 8개 + 500,000 FE 소비
* 매 충전마다 고정적으로 초고전압 번개 1개 산출, 매트릭스는 소비되지 않음

## 작동 흐름

1. GUI 버튼으로 모드 선택
2. 재료 준비 — 고전압 모드는 과부하 수정 가루를 넣고; 초고전압 모드는 매트릭스가 장착되어 있고 네트워크에 충분한 고전압 번개가 있는지 확인
3. FE 에너지 연결(코일 내부 FE 버퍼 16,000,000 FE, 최대 수용 속도 200,000 FE/tick)
4. 코일이 현재 모드를 잠그고 충전 시작(약 5 tick)
5. 충전 완료 후, 산물이 ME 네트워크에 일괄 주입

## 번개 붕괴 매트릭스

<ItemImage id="ae2lt:lightning_collapse_matrix" scale="2" float="left" />

초고전압 모드는 코일의 매트릭스 슬롯에 **번개 붕괴 매트릭스** 한 개를 반드시 장착해야 합니다. 매트릭스는 가공 과정에서 **소비되지 않지만**, 슬롯에 항상 남아 있어야 합니다.
