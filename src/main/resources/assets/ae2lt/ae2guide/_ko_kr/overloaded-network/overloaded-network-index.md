# 과부하 네트워크

AE2 Lightning Tech는 대형 기지에서 바닐라 AE2의 채널 및 용량 제한을 돌파하기 위한 「과부하」 네트워크 설비 세트를 제공합니다.

바닐라 대비:

* 각 **과부하 ME 제어기**가 네트워크에 추가로 128 채널 제공
* 과부하 케이블 자체에는 채널 상한이 없으며, 전체 과부하 네트워크는 제어기 총수에 따라 전역적으로 채널을 할당
* 과부하 패턴 공급기와 과부하 ME 인터페이스는 36개의 슬롯을 가지며, 거리에 걸친 무선 연결 지원
