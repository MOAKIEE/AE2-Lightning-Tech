# 과부하 패턴 공급기

<Row>
  <BlockImage id="ae2lt:overloaded_pattern_provider" scale="4" />
</Row>

**과부하 패턴 공급기**는 바닐라 <ItemLink id="ae2:pattern_provider" />의 강화 버전으로, **36개의 패턴 슬롯**을 가지고 있으며, **무선 모드**를 지원하여 거리에 걸쳐 원격 기계로 가공 재료를 분배할 수 있습니다.

## 핵심 특성

* **36개의 패턴 슬롯**(바닐라는 9개)
* **두 가지 작동 모드**: 일반 모드와 무선 모드
* **과부하 패턴 지원**: 바닐라 패턴 외에도 이 모드의 과부하 패턴 지원
* **자동 회수**: 원격 기계에서 가공 산물을 자동으로 회수
* **입력 필터링**: 패턴 레시피가 정의한 산물만 ME 네트워크로 반환 허용

## 작동 모드

### 일반 모드

일반 모드에서 과부하 패턴 공급기는 재료를 **물리적으로 인접한** 기계로 추송하며, 동작은 바닐라 패턴 공급기와 동일합니다. 주요 차이점은 더 많은 패턴 슬롯을 가진 것입니다.

### 무선 모드

<ItemImage id="ae2lt:overloaded_wireless_connect_tool" scale="2" float="left" />

무선 모드에서 과부하 패턴 공급기는 거리에 걸쳐 재료를 원격 기계로 분배할 수 있습니다. 무선 연결을 설정하려면 **과부하 무선 연결 도구**를 사용:

1. 과부하 무선 연결 도구를 들고, 과부하 패턴 공급기를 **Shift + 우클릭**으로 선택
2. 다음 대상 기계의 특정 면을 **우클릭**으로 연결 설정
3. 동일한 공급기는 여러 원격 기계를 바인딩 가능

연결 완료 후, 공급기는 선택된 분배 전략에 따라 원격 기계로 재료를 분배합니다.

### 분배 전략

| 전략 | 설명 |
|------|------|
| 순차 분배 | 순서대로 원격 기계마다 하나씩 분배 |
| 균등 분배 | 연결된 모든 원격 기계에 재료를 균등하게 분배 |

## 회수 모드

회수 모드는 가공 완료 후 산물이 원격 기계에서 어떻게 회수되는지를 결정합니다:

| 모드 | 설명 |
|------|------|
| 끔 | 자동 회수 안 함 |
| 자동 | 원격 기계에서 능동적으로 산물 추출 |
| 추출 | 원격 기계가 가상 출력 포트로 산물을 밀어넣음; 공급기는 수동 수신 |

## 속도 모드

| 속도 단계 | 설명 |
|-------|------|
| 일반 | 표준 쿨다운 시간(5 ~ 80 tick) |
| 고속(프로브) | 적응형 쿨다운, 프로브 메커니즘으로 준비 상태를 미리 감지(1 ~ 40 tick) |

## 입력 필터링

「입력 필터링」 활성화 후, 공급기는 산물 회수 시 현재 패턴에 정의된 출력 아이템만 수용하여 잘못된 아이템이 네트워크로 들어가는 것을 방지합니다.

## 과부하 패턴과 과부하 패턴 인코더

<Row>
  <ItemImage id="ae2lt:overload_pattern" scale="2" />
  <ItemImage id="ae2lt:overload_pattern_encoder" scale="2" />
</Row>

**과부하 패턴**은 전용 패턴이며, **과부하 패턴 공급기에서만 사용할 수 있습니다**. **과부하 패턴 인코더**로 인코딩합니다.

과부하 패턴 인코더 지원 기능:

* 주 입력과 주 출력 설정
* 부산물 설정
* **NBT 무시** 스위치: 활성화 시 아이템 매칭에서 NBT 데이터를 무시

## 자동화 권장사항

* 대규모 자동 합성 시스템에서, 무선 모드는 배선 복잡도를 크게 줄일 수 있음
* 「균등 분배」 전략과 여러 가공 기계를 조합하여 병렬 가공 구현 가능
* 응답성 향상을 위해 「고속」 속도 단계 활성화 권장
* 자동화 시나리오에 따라 적절한 회수 모드 선택
