# 번개 어셈블리 챔버

<Row>
  <BlockImage id="ae2lt:lightning_assembly_chamber" scale="4" />
</Row>

**번개 어셈블리 챔버**는 입력 수량이 많은 고급 레시피를 처리하는 데 사용됩니다. 번개 시뮬레이션 룸의 입력 슬롯 3개에 비해, 번개 어셈블리 챔버는 9개의 입력 슬롯을 가지고 있어 다양한 종류의 재료 조합이 필요한 복잡한 조립 작업에 적합합니다.

## 슬롯과 용량

| 슬롯 | 용량 | 설명 |
|------|------|------|
| 입력 슬롯 ×9 | 8,192 | 다양한 원료를 넣으며, 슬롯 순서는 구분하지 않음 |
| 매트릭스 슬롯 ×1 | 1 | 번개 붕괴 매트릭스를 선택적으로 장착 |
| 출력 슬롯 ×1 | 8,192 | 가공 산물; 오직 기계 쓰기만 허용 |
| 가속 카드 슬롯 | — | AE2 가속 카드 최대 4장 장착 |

## 작동 흐름

1. 9개 입력 슬롯에 원료를 각각 배치
2. 기계가 현재 입력에 따라 사용 가능한 레시피 매칭
3. 레시피가 매칭되고 ME 네트워크에 번개와 FE 에너지가 충분하면, 기계는 레시피를 잠그고 가공 시작
4. 가공 과정에서 번개(레시피에 따라 고전압 또는 초고전압)와 FE 에너지를 동시에 소비
5. 가공 완료 후, 산물은 출력 슬롯으로 진입

## 번개 시뮬레이션 룸과의 비교

| 특성 | 번개 시뮬레이션 룸 | 번개 어셈블리 챔버 |
|------|-----------|-----------|
| 입력 슬롯 수 | 3 | 9 |
| 슬롯당 최대 스택 | 8,192 | 8,192 |
| 적용 시나리오 | 단순 재료 변환 | 복잡한 다중 재료 조립 |
| 매트릭스 대체 | 지원 | 지원 |
| 가속 카드 | 최대 4장 | 최대 4장 |

## 매트릭스 대체

<ItemImage id="ae2lt:lightning_collapse_matrix" scale="2" float="left" />

번개 시뮬레이션 룸과 마찬가지로, 번개 어셈블리 챔버도 **매트릭스 대체**를 지원합니다. **번개 붕괴 매트릭스** 장착 후, 일부 원래 **초고전압 번개**를 요구하는 레시피는 몇 배 수량의 **고전압 번개**를 대신 사용할 수 있습니다.

## 자동 배출

자동 배출 활성화 후, 가공 완료된 산물은 허용된 방향의 인접 컨테이너로 자동 추송됩니다. 구체적인 방향은 GUI의 「출력면 설정」 인터페이스에서 설정합니다.
