# 수정 촉매기

<Row>
  <BlockImage id="ae2lt:crystal_catalyzer" scale="4" />
</Row>

**수정 촉매기**는 물, FE, 슬롯 내 재료를 입력으로 하는 전용 가공에 사용됩니다. **수정 모드**와 **가루화 모드** 두 가지 작동 모드가 있으며, 동일한 기계 내에서 전환할 수 있습니다.

## 슬롯과 용량

| 슬롯 | 용량 | 설명 |
|------|------|------|
| 촉매제 슬롯 | 256 | 현재 모드에 필요한 아이템을 넣음; 슬롯 내 아이템은 가공 중 **소비되지 않음** |
| 매트릭스 슬롯 | 1 | 번개 붕괴 매트릭스를 선택적으로 장착하여 산출량 증가 |
| 출력 슬롯 | 1,024 | 가공 산물; 오직 기계 쓰기만 허용, 외부 입력 거부 |
| 유체 슬롯 | 16,000 mB | 파이프로 물을 주입; 각 회 1,000 mB 고정 소비 |
| FE 에너지 버퍼 | 1,000,000 FE | 내장 에너지 버퍼 |

## 작동 모드

| 모드 | 용도 | 입력 예 | 출력 예 |
|------|------|----------|----------|
| 수정 모드 | 해당 결정 블록에서 수정 추출 | 서투스 석영 블록, 액정 블록, 과부하 수정 블록 | 서투스 석영 수정, 액정 수정, 과부하 수정 |
| 가루화 모드 | 해당 결정 블록에서 수정 가루 추출 | 서투스 석영 블록, 액정 블록, 과부하 수정 블록 | 서투스 석영 가루, 액정 가루, 과부하 수정 가루 |

두 모드는 동일한 촉매제 슬롯, 유체 슬롯, 출력 슬롯을 공유합니다. 촉매제 슬롯의 아이템은 레시피 매칭과 병렬 수량 계산에만 사용되며, 소비되지 않습니다.

## 작동 흐름

1. GUI 왼쪽 버튼에서 수정 모드 또는 가루화 모드 선택
2. 유체 파이프로 유체 슬롯에 물 주입
3. 현재 모드에 해당하는 아이템을 촉매제 슬롯에 배치
4. FE 에너지 연결
5. 기계가 사용 가능한 레시피 매칭 후 자동 가공
6. 완성된 산물은 출력 슬롯으로 진입

## 병렬과 물 소비

각 회 가공마다 **1,000 mB의 물**을 고정 소비하며, 병렬 수량이나 매트릭스 보너스로 인해 증가하지 않습니다.

촉매제 슬롯의 재고가 병렬 횟수를 결정합니다: 병렬 횟수 = 슬롯 내 수량 ÷ 레시피 요구 수량. 현재 내장 레시피는 매번 해당 블록 1개가 필요하므로, 현재 모드에 맞는 64개의 블록을 넣으면 매 회 64회 병렬로 산출량이 계산됩니다.

병렬 수량은 레시피 시작 시 고정됩니다; 가공 과정에서 슬롯 내 아이템을 계속 가감해도 이미 고정된 이 회의 산출량은 변경되지 않습니다.

## 번개 붕괴 매트릭스 보너스

<ItemImage id="ae2lt:lightning_collapse_matrix" scale="2" float="left" />

매트릭스 슬롯에 **번개 붕괴 매트릭스**가 장착된 경우, 수정 촉매기의 단회 산출 수량이 **4배**로 증가합니다. 매트릭스는 가공 중 소비되지 않습니다.

최종 산출량 = 기본 산출량 × 병렬 횟수 × 매트릭스 배율.

## 기타 설명

* 수정 촉매기는 **외부 FE 에너지 입력**으로 전력을 공급받으며, ME 네트워크에서 AE 에너지를 직접 추출하지 않습니다
* 기계 본체도 ME 네트워크 설비이므로, AE2 인터페이스나 패턴 공급기를 통한 자동 공급을 위해서는 네트워크에 연결되어야 합니다
* 자동 배출 기능을 지원하며, 산물 방향은 GUI에서 설정할 수 있습니다
* 수정 촉매기는 가속 카드를 **지원하지 않습니다**
