# 과부하 처리 공장

<Row>
  <BlockImage id="ae2lt:overload_processing_factory" scale="4" />
</Row>

**과부하 처리 공장**은 이 모드에서 규모가 가장 크고 처리량이 가장 높은 가공 설비입니다. 아이템과 유체의 혼합 입출력을 지원하며, 번개 붕괴 매트릭스를 통해 다중 병렬 가공을 해금하여, 대규모 산업화 생산 라인의 핵심 기계입니다.

## 슬롯과 용량

| 슬롯/구성 요소 | 용량 | 설명 |
|------------|------|------|
| 아이템 입력 슬롯 ×9 | 8,192 | 가공할 아이템을 배치 |
| 매트릭스 슬롯 ×1 | 32 | 번개 붕괴 매트릭스를 장착하여 병렬 능력 해금 |
| 아이템 출력 슬롯 ×1 | 8,192 | 가공 산물; 오직 기계 쓰기만 허용 |
| 유체 입력 슬롯 ×1 | 512,000 mB | 파이프로 필요한 유체 주입 |
| 유체 출력 슬롯 ×1 | 512,000 mB | 가공으로 산출된 유체 저장 |
| FE 에너지 버퍼 | 640,000,000 FE(기본값) | 내장 에너지 버퍼 |
| 가속 카드 슬롯 | — | AE2 가속 카드 최대 4장 장착 |

## 작동 흐름

1. 아이템 입력 슬롯과 유체 입력 포트를 통해 원료 공급
2. 기계가 사용 가능한 레시피 매칭
3. 레시피가 매칭되고 ME 네트워크의 번개와 내부 FE 에너지가 충분하면, 기계는 가공 시작
4. 가공 과정에서 번개와 FE 에너지를 지속적으로 소비
5. 산물은 각각 아이템 출력 슬롯과 유체 출력 슬롯으로 진입

## 다중 병렬 가공

<ItemImage id="ae2lt:lightning_collapse_matrix" scale="2" float="left" />

과부하 처리 공장의 병렬 능력은 장착된 **번개 붕괴 매트릭스** 수량에 의해 결정됩니다. 매트릭스 슬롯에는 최대 32개의 매트릭스를 수용할 수 있으며, 각 매트릭스는 설정에 따라 일정량의 병렬 상한을 제공합니다(기본값 각 8회 병렬, 상한 32 × 8 = 256회):

* 매트릭스 미장착 시, 공장은 병렬 가공이 불가능
* 더 많은 매트릭스 장착으로 병렬 상한을 선형적으로 향상
* 실제 병렬 수량은 입력 재료 수량과 레시피의 단회 사용량에 의해 동시에 제한됨
* 병렬 횟수가 많아질수록 단회 가공의 산출이 많아지며, 동시에 FE 에너지 소비도 상응하게 선형 증가

## 가속 카드와 에너지 소비

과부하 처리 공장은 최대 4장의 가속 카드를 지원합니다. 가속 카드 한 장이 추가될 때마다, tick당 소비할 수 있는 FE 상한이 크게 향상됩니다:

| 가속 카드 수 | tick당 FE 상한(기본값) |
|-----------|-------------------------|
| 0 | 400,000 FE |
| 1 | 2,000,000 FE |
| 2 | 8,000,000 FE |
| 3 | 32,000,000 FE |
| 4 | 128,000,000 FE |

## 유체 입력 / 출력

유체 슬롯은 임의의 유체 파이프 시스템을 통해 입력과 출력이 가능합니다. 입력 슬롯은 레시피가 요구하는 유체 유형을 엄격히 검증합니다.

## 자동 배출

자동 배출 활성화 후, 아이템 산물은 허용된 방향의 인접 컨테이너로 자동 추송됩니다. 구체적인 방향은 GUI의 「출력면 설정」 인터페이스에서 설정합니다.

## 에너지 공급

과부하 처리 공장은 **외부 FE 에너지 입력**으로 전력을 공급받으며, ME 네트워크에서 AE 에너지를 직접 추출하지 않습니다. FE 에너지를 출력할 수 있는 임의의 파이프/도선으로 기계 측면에 에너지를 연결하세요.
