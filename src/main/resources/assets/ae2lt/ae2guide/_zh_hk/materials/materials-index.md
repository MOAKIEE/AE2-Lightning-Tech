# 材料與物品

AE2 閃電科技引入了大量新材料。以下頁面分別詳細介紹了各個核心材料與物品。

<SubPages />

## 天氣凝核

天氣凝核由 [大氣電離儀](../machines/atmospheric-ionizer.md) 消耗，用於控制世界天氣。可在閃電模擬室、閃電裝配室或過載處理工廠中生產。

<ItemGrid>
  <ItemIcon id="ae2lt:clear_condensate" />
  <ItemIcon id="ae2lt:rain_condensate" />
  <ItemIcon id="ae2lt:thunderstorm_condensate" />
</ItemGrid>

| 物品 | 目標天氣 | AE 消耗 |
|------|---------|---------|
| 晴空凝核 | 晴天 | 500,000 AE |
| 降雨凝核 | 雨天 | 1,000,000 AE |
| 雷暴凝核 | 雷暴 | 8,000,000 AE |
