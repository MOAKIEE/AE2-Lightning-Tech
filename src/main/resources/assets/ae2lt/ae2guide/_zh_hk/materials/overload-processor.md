# 過載處理器

**過載處理器**是一種高級處理器，用於製作大多數過載機器和高級設備。其生產過程需要透過 AE2 壓印器進行多步合成。

## 合成流程

1. 取得**過載壓印模板**（透過閃電轉化、閃電模擬室或過載處理工廠獲得）
2. 在 AE2 壓印器中使用過載壓印模板與過載合金錠粗胚，壓印出**未過載的電路板**
3. 將未過載的電路板放回壓印器，加工為**過載電路板**
4. 將過載電路板與其他材料在 AE2 分子裝配室中組合，即可獲得**過載處理器**

## 物品一覽

| 物品 | 作用 |
|------|------|
| 過載壓印模板 | 在 AE2 壓印器中使用的壓印模板；透過閃電轉化或機器加工獲得 |
| 未過載的電路板 | 中間產物；用過載壓印模板壓印過載合金錠粗胚得到 |
| 過載電路板 | 成品電路板；由未過載的電路板在壓印器中進一步加工得到 |
| 過載處理器 | 最終處理器，用於機器和設備的合成配方 |
