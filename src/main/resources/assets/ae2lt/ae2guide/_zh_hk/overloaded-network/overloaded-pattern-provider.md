# 過載樣板供應器

<Row>
  <BlockImage id="ae2lt:overloaded_pattern_provider" scale="4" />
</Row>

**過載樣板供應器**是原版 <ItemLink id="ae2:pattern_provider" /> 的增強版本，擁有 **36 個樣板槽位**，並支援**無線模式**，可跨距離向遠端機器派發加工材料。

## 核心特性

* **36 個樣板槽位**（原版為 9 個）
* **兩種工作模式**：普通模式與無線模式
* **支援過載樣板**：除原版樣板外，還支援本模組的過載樣板
* **自動回收**：從遠端機器自動取回加工產物
* **輸入過濾**：僅允許樣板配方定義的產物返回 ME 網絡

## 工作模式

### 普通模式

普通模式下，過載樣板供應器將材料推送到**物理相鄰**的機器中，行為與原版樣板供應器一致。主要差異是擁有更多樣板槽位。

### 無線模式

<ItemImage id="ae2lt:overloaded_wireless_connect_tool" scale="2" float="left" />

無線模式下，過載樣板供應器可以跨距離將材料派發到遠端機器。建立無線連接需要使用**過載無線連接工具**：

1. 手持過載無線連接工具，**Shift + 右擊**過載樣板供應器以完成選中
2. 再對目標機器的特定面**右擊**以建立連接
3. 同一台供應器可以綁定多台遠端機器

連接完成後，供應器將按所選分配策略向遠端機器派發材料。

### 分配策略

| 策略 | 說明 |
|------|------|
| 逐台發配 | 按順序逐一向每台遠端機器派發 |
| 均分發配 | 將材料均勻分配給所有連接的遠端機器 |

## 回收模式

回收模式決定了加工完成後產物如何從遠端機器取回：

| 模式 | 說明 |
|------|------|
| 關 | 不進行自動回收 |
| 自動 | 主動從遠端機器中提取產物 |
| 彈出 | 遠端機器將產物彈入虛擬輸出口；供應器被動接收 |

## 速度模式

| 速度檔 | 說明 |
|-------|------|
| 普通 | 標準冷卻時間（5 ~ 80 tick） |
| 快速（探針） | 自適應冷卻，使用探針機制提前檢測就緒狀態（1 ~ 40 tick） |

## 輸入過濾

啟用「輸入過濾」後，供應器在回收產物時只接收當前樣板已定義的輸出物品，防止錯誤的物品進入網絡。

## 過載樣板與過載樣板編碼器

<Row>
  <ItemImage id="ae2lt:overload_pattern" scale="2" />
  <ItemImage id="ae2lt:overload_pattern_encoder" scale="2" />
</Row>

**過載樣板**是一種專用樣板，**僅可在過載樣板供應器中使用**。使用**過載樣板編碼器**進行編碼。

過載樣板編碼器支援：

* 設定主輸入與主輸出
* 設定副產物
* **忽略 NBT** 開關：啟用後匹配物品時忽略 NBT 資料

## 自動化建議

* 大規模自動合成系統中，無線模式可大幅減少布線複雜度
* 配合「均分發配」策略與多台加工機器，可實現並行加工
* 建議啟用「快速」速度檔以提升回應性
* 根據自動化場景選擇合適的回收模式
