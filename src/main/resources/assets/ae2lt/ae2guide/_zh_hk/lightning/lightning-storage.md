# 閃電儲存元件

閃電儲存元件是專用於在 ME 網絡中存放閃電的儲存介質。與普通物品 / 流體儲存元件一樣，只需放入 <ItemLink id="ae2:drive" /> 即可使用。

## 元件等級

閃電儲存元件共有五檔，容量逐級遞增。每枚元件可以同時存放**高壓閃電**與**極高壓閃電**兩種類型。

  <ItemIcon id="ae2lt:lightning_storage_component_i" />
  <ItemIcon id="ae2lt:lightning_storage_component_ii" />

| 元件等級 | 有效容量 | 空閒耗電 |
|---------|---------|---------|
| 閃電儲存元件 I | 256 | 32 AE/t |
| 閃電儲存元件 II | 1,024 | 128 AE/t |
| 閃電儲存元件 III | 4,096 | 512 AE/t |
| 閃電儲存元件 IV | 16,384 | 2,048 AE/t |
| 閃電儲存元件 V | 65,536 | 8,192 AE/t |

## 無限儲存元件

<ItemImage id="ae2lt:infinite_storage_cell" scale="2" float="left" />

**無限儲存元件**沒有容量上限，內部允許任意數量、任意類型的物品共存。受合成成本與工藝限制，**無法量產**。
