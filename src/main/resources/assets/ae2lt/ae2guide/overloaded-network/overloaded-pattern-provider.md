# 过载样板供应器

<Row>
  <BlockImage id="ae2lt:overloaded_pattern_provider" scale="4" />
</Row>

**过载样板供应器**是原版 <ItemLink id="ae2:pattern_provider" /> 的大幅增强版本，拥有 **36 个样板槽位**，并支持**无线模式**，可以将材料直接发配到远程机器而无需物理相邻。

## 核心特性

* **36 个样板槽位**（原版为 9 个）
* **两种工作模式**：普通模式和无线模式
* **支持过载样板**：可使用专用的过载样板
* **自动回收**功能：从远程机器自动取回产物
* **输入过滤**：仅允许样板配方中的产物返回

## 工作模式

### 普通模式

在普通模式下，过载样板供应器的行为与原版样板供应器一致——将材料推送到**物理相邻**的机器中。区别仅在于拥有更多的样板槽位。

### 无线模式

<ItemImage id="ae2lt:overloaded_wireless_connect_tool" scale="2" float="left" />

在无线模式下，过载样板供应器可以跨越距离将材料发送到远程机器。要建立无线连接，需要使用**过载无线连接工具**：

1. 手持过载无线连接工具，**Shift + 右击**过载样板供应器进行选中
2. 然后**右击**目标机器的特定面来建立连接
3. 可以为同一个供应器绑定多台远程机器

连接成功后，样板供应器会根据分配策略将材料无线发送到远程机器。

### 分配策略

无线模式下有两种分配策略：

| 策略 | 说明 |
|------|------|
| 逐台发配 | 按顺序向每台机器逐一发送 |
| 均分发配 | 将材料均匀分配给所有连接的机器 |

## 回收模式

回收模式决定了加工完成后如何将产物从远程机器取回：

| 模式 | 说明 |
|------|------|
| 关 | 不进行自动回收 |
| 自动 | 主动从远程机器提取产物 |
| 弹出 | 远程机器自动将产物弹入虚拟输出口，供应器自动接收 |

## 速度模式

| 模式 | 说明 |
|------|------|
| 普通 | 标准冷却时间（5~80 tick） |
| 快速（探针） | 自适应冷却，使用探针机制提前检测就绪状态（1~40 tick） |

## 输入过滤

开启「输入过滤」后，供应器在回收产物时只接收当前样板中已定义的输出物品，可以防止错误的物品进入网络。

## 自动化建议

* 对于大规模自动合成系统，无线模式可以大幅减少布线复杂度
* 搭配均分发配策略和多台加工机器，实现并行加工
* 建议开启「快速」速度模式以获得最佳响应时间
* 建议在全自动化场景中选择合适的回收模式

## 过载样板与过载样板编码器

<Row>
  <ItemImage id="ae2lt:overload_pattern" scale="2" />
  <ItemImage id="ae2lt:overload_pattern_encoder" scale="2" />
</Row>

**过载样板**是一种特殊样板，**只能用于过载样板供应器**。使用**过载样板编码器**来编码过载样板。

过载样板编码器支持：
* 设置输入和输出
* 设置副产物
* **忽略 NBT** 开关：开启后在匹配物品时忽略 NBT 数据
