# 过载网络

AE2 闪电科技引入了一系列「过载」网络设备，突破了原版 AE2 的频道和容量限制，为大型基地提供更强大的 ME 网络基础设施。

## 过载网络设备

* [过载 ME 控制器](overloaded-controller.md) — 更大功率的网络控制器
* [过载线缆](overloaded-cable.md) — 128 频道的超级线缆
* [过载样板供应器](overloaded-pattern-provider.md) — 支持无线模式的大容量样板供应器
* [过载 ME 接口](overloaded-interface.md) — 36 槽位的超级接口，支持无线 I/O
