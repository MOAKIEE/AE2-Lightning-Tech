# Materials & Items

AE2 Lightning Tech introduces a large set of new materials. Below is a quick reference for the key ones.

## Basic Materials

<ItemGrid>
  <ItemIcon id="ae2lt:overload_crystal" />
  <ItemIcon id="ae2lt:overload_crystal_dust" />
  <ItemIcon id="ae2lt:electro_chime_crystal" />
  <ItemIcon id="ae2lt:perfect_electro_chime_crystal" />
</ItemGrid>

| Item | Notes |
|------|-------|
| Overload Crystal | Core starter material, harvested from Overload Crystal Clusters |
| Overload Crystal Dust | Dropped from buds that are not yet fully grown; consumed by the Tesla Coil in HV mode |
| Electro-Chime Crystal | Catalyst for the Lightning Collector; cultivated gradually by natural lightning strikes |
| Perfect Electro-Chime Crystal | The final form reached once cultivation is maxed; provides a fixed maximum-yield bonus |
