# 过载处理工厂

<Row>
  <BlockImage id="ae2lt:overload_processing_factory" scale="4" />
</Row>

**过载处理工厂**是本模组中最大、最强力的加工设备。它支持物品和流体的混合输入，具备多并行加工能力，是大规模工业化产线中不可或缺的核心机器。

## 基本结构

过载处理工厂具有以下槽位和组件：

* **物品输入槽**（9 个）：放入待加工的物品，单槽最大堆叠 8192
* **物品输出槽**（1 个）：加工完成的物品产物
* **矩阵槽**（1 个）：安装闪电坍缩矩阵以解锁并行加工能力（最多 64 个）
* **流体输入槽**（1 个，512,000 mB 容量）：输入所需的流体
* **流体输出槽**（1 个，512,000 mB 容量）：加工完成的流体产物
* **FE 能量缓冲**：内置能量缓存（容量可配置）
* **速度卡槽**：可安装 AE2 速度卡（最多 4 张）

## 工作流程

1. 通过物品输入槽和流体输入口提供原料
2. 机器会自动匹配适合的加工配方
3. 若配方匹配且内部 FE 能量充足，机器开始加工
4. 加工过程中持续消耗 FE 能量
5. 产物会出现在对应的物品/流体输出槽中

## 多并行加工

<ItemImage id="ae2lt:lightning_collapse_matrix" scale="2" float="left" />

过载处理工厂的并行加工能力由安装的**闪电坍缩矩阵**数量决定。矩阵槽最多可放入 64 个矩阵，每个矩阵提供一定的并行倍率（可配置）。

* 未安装矩阵时，工厂无法进行并行加工
* 安装更多矩阵可以线性提升并行上限
* 实际并行数量还取决于输入材料的数量和配方的单次用量
* 并行倍率越高，单次加工的产出越多，但能耗也会相应增加

## 流体输入/输出

过载处理工厂可以通过管道系统（或其他流体传输手段）向流体输入槽注入流体。每个流体槽的容量为 512,000 mB。

流体输出同样可以通过管道抽取。

## 自动弹出

与其他机器类似，过载处理工厂支持自动弹出功能。可以为物品产出配置自动弹出方向。

## FE 能量供应

过载处理工厂通过**外部 FE 能量输入**来供电，而非从 ME 网络直接提取 AE 能量。请使用任何能够提供 FE 能量的管道/导线将能量输入到机器侧面。

## 自动化建议

* 使用 AE2 样板供应器或过载样板供应器配合液体和物品样板
* 通过流体管道系统为流体输入槽供料
* 安装闪电坍缩矩阵以解锁并行加工，提高单机产出
* 安装速度卡以加快加工速度（最多 4 张）
