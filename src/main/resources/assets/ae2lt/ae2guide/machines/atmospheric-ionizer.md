# 大气电离仪

<Row>
  <BlockImage id="ae2lt:atmospheric_ionizer" scale="4" />
</Row>

**大气电离仪**是一台天气控制设备。它通过消耗**天气凝核**和 AE 能量来强制改变世界的天气状态，让你不再需要等待自然雷暴。

## 天气凝核

天气凝核有三种类型，分别对应三种天气：

<ItemGrid>
  <ItemIcon id="ae2lt:clear_condensate" />
  <ItemIcon id="ae2lt:rain_condensate" />
  <ItemIcon id="ae2lt:thunderstorm_condensate" />
</ItemGrid>

| 凝核类型 | 目标天气 | AE 消耗 | 持续时间 |
|---------|---------|---------|---------|
| 晴空凝核 | 晴天 | 500,000 AE | 12,000 ~ 180,000 tick |
| 降雨凝核 | 雨天 | 1,000,000 AE | 12,000 ~ 24,000 tick |
| 雷暴凝核 | 雷暴 | 8,000,000 AE | 3,600 ~ 15,600 tick |

## 工作流程

1. 将大气电离仪连接到 ME 网络
2. 将所需的天气凝核放入输入槽
3. 机器会自动从 ME 网络中提取 AE 能量进行「电离」
4. 电离完成后，天气将被强制切换为目标天气
5. 凝核在使用后会被消耗

## GUI 状态说明

| 状态 | 含义 |
|------|------|
| 待机 | 无凝核输入 |
| 电离中 | 正在消耗 AE 能量 |
| 等待 AE | ME 网络 AE 能量不足 |
| 等待凝核 | 输入槽中没有凝核 |
| 当前维度不支持天气 | 所在维度无法控制天气（如末地） |
| 目标天气已生效 | 目标天气已经是当前天气 |
| 准备就绪 | 电离完成，即将切换天气 |

## 注意事项

* 大气电离仪消耗的是 **AE 能量**（来自 ME 网络），而非 FE 能量
* 雷暴凝核的 AE 消耗最高（8,000,000 AE），请确保网络有足够的能量供应
* 部分维度不支持天气变化（如末地、下界），在这些维度中电离仪无法工作
* 当目标天气已经是当前天气时，机器不会消耗凝核

## 自动化建议

电离仪非常适合与闪电收集器配合使用：

1. 使用 AE2 等级发射器监控网络中的闪电存量
2. 当闪电存量低于阈值时，向电离仪输入雷暴凝核
3. 雷暴天气触发后，闪电收集器即可开始收集自然雷击
