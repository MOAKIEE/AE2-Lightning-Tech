# 過載合金

**過載合金**是一種高級金屬材料，廣泛用於機器框架、過載控制器以及多種終末級組件。

## 加工流程

1. **過載合金錠粗胚** — 在工作台直接合成
2. **過載合金錠** — 成品合金錠，可在閃電模擬室中由粗胚精煉，也可在過載處理工廠中由金屬與晶體類材料批量生產
3. **過載合金板** — 在閃電模擬室或過載處理工廠中由過載合金錠壓製而成，是極限過載核心等頂級配方所需的材料

過載合金錠可以由兩種機器獲得：

| 機器 | 輸入 | 閃電 | 能量 | 產出 |
|------|------|------|------|------|
| 閃電模擬室 | `1` 過載合金錠粗胚 + `4` 過載水晶粉 | `1` 高壓閃電 | `20,000` | `1` 過載合金錠 |
| 過載處理工廠 | `1` 銅塊 + `1` 鐵塊 + `1` 金塊 + `1` 紫水晶塊 + `1` 終界之眼 + `1` 紫頌果 | `1` 高壓閃電 | `100,000` | `27` 過載合金錠 |

## 物品一覽

| 物品 | 作用 |
|------|------|
| 過載合金錠粗胚 | 中間產物 |
| 過載合金錠 | 成品合金錠，大多數過載機器的關鍵合成材料 |
| 過載合金板 | 壓製板材，用於極限過載核心等高級配方 |
