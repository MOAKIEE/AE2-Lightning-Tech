# 加工機器

AE2 閃電科技提供了多種加工機器，分別應對不同類型的材料合成需求。除大氣電離儀消耗 AE 能量之外，其餘機器均接入 ME 網路同時使用外部 FE 能量，並在加工時從 ME 網路中提取閃電。

<SubPages />

## 通用特性

### 自動彈出

多數機器支援**自動彈出**。啟用後，機器完成加工時會將產物推送到允許方向的相鄰容器。具體方向可在機器 GUI 的「設定輸出面」介面中設定。

### 速度卡

除水晶催化器外，其餘加工機器均支援安裝 AE2 原版的**速度卡**以提升加工速度，最多可安裝 4 張。

### 閃電代償

部分需要極高壓閃電的配方支援**矩陣代償**機制：當機器內安裝了**閃電坍縮矩陣**時，可以使用數倍量的高壓閃電替代極高壓閃電完成加工。具體代償比例取決於配方本身。
