# 過載處理工廠

<Row>
  <BlockImage id="ae2lt:overload_processing_factory" scale="4" />
</Row>

**過載處理工廠**是本模組規模最大、吞吐最高的加工設備。它支援物品與流體的混合輸入輸出，並透過閃電坍縮矩陣解鎖多並行加工，是大規模工業化產線的核心機器。

## 槽位與容量

| 槽位 / 組件 | 容量 | 說明 |
|------------|------|------|
| 物品輸入槽 ×9 | 8,192 | 放入待加工的物品 |
| 矩陣槽 ×1 | 32 | 安裝閃電坍縮矩陣以解鎖並行能力 |
| 物品輸出槽 ×1 | 8,192 | 加工產物；僅允許機器寫入 |
| 流體輸入槽 ×1 | 512,000 mB | 透過管道注入所需流體 |
| 流體輸出槽 ×1 | 512,000 mB | 存放加工產出的流體 |
| FE 能量緩衝 | 640,000,000 FE（預設） | 內建能量緩存 |
| 速度卡槽 | — | 最多安裝 4 張 AE2 速度卡 |

## 工作流程

1. 透過物品輸入槽和流體輸入口提供原料
2. 機器匹配可用配方
3. 配方匹配且 ME 網路中閃電與內部 FE 能量充足後，機器開始加工
4. 加工過程中持續消耗閃電與 FE 能量
5. 產物分別進入物品輸出槽和流體輸出槽

## 多並行加工

<ItemImage id="ae2lt:lightning_collapse_matrix" scale="2" float="left" />

過載處理工廠的並行能力由安裝的**閃電坍縮矩陣**數量決定。矩陣槽最多容納 32 枚矩陣，每枚矩陣按設定提供一定量的並行上限（預設每枚 8 次並行，上限 32 × 8 = 256 次）：

* 未安裝矩陣時，工廠無法並行加工
* 安裝更多矩陣可線性提升並行上限
* 實際並行數量同時受限於輸入材料數量與配方的單次用量
* 並行次數越多，單次加工的產出越多，同時 FE 能量消耗相應線性增加

## 速度卡與能量消耗

過載處理工廠支援最多 4 張速度卡。每多裝一張速度卡，單 tick 可消耗的 FE 上限即顯著提升：

| 速度卡數量 | 單 tick FE 上限（預設值） |
|-----------|-------------------------|
| 0 | 400,000 FE |
| 1 | 2,000,000 FE |
| 2 | 8,000,000 FE |
| 3 | 32,000,000 FE |
| 4 | 128,000,000 FE |

## 流體輸入 / 輸出

流體槽可透過任意流體管道系統進行輸入與輸出。輸入槽會嚴格校驗配方所需的流體類型。

## 自動彈出

啟用自動彈出後，物品產物會被自動推送到允許方向的相鄰容器。具體方向在 GUI 的「設定輸出面」介面中設定。

## 能量供應

過載處理工廠透過**外部 FE 能量輸入**供能，並不從 ME 網路直接提取 AE 能量。請使用任意能夠輸出 FE 能量的管道 / 導線將能量接入機器側面。
