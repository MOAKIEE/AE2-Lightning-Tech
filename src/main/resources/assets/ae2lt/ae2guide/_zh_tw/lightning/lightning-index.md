# 閃電系統

閃電是本模組向 ME 網路加入的一種全新儲存類型。它與物品、流體一樣可以在終端中查看數量、存入儲存元件、並由加工機器消耗。

## 兩個等級

| 等級 | 產生方式 |
|------|---------|
| **高壓閃電** | 閃電收集器被**人工閃電**命中（玩家攜帶過載水晶吸引，或特斯拉線圈放電） |
| **極高壓閃電** | 閃電收集器被**自然閃電**命中（雷暴天氣下的天然雷電），或由特斯拉線圈從高壓閃電升壓而來 |

多數基礎配方僅需要高壓閃電，部分高級配方需要極高壓閃電。安裝**閃電坍縮矩陣**時，部分極高壓配方可以使用數倍量的高壓閃電進行代償。
