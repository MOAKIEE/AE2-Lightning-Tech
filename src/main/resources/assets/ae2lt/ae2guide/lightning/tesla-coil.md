# 特斯拉线圈

<Row>
  <BlockImage id="ae2lt:tesla_coil" scale="4" />
</Row>

**特斯拉线圈**是稳定、大规模生产闪电的核心设备。与依赖天气的闪电收集器不同，特斯拉线圈通过消耗 FE 能量和材料来主动生产闪电。

## 工作模式

特斯拉线圈有两种工作模式，可以在 GUI 中切换：

### 高压模式

* **输入**：过载水晶粉 + FE 能量
* **输出**：高压闪电（存入 ME 网络）
* 支持批量处理——放入更多的过载水晶粉可以在一次充能中生产更多的闪电

### 极高压模式

* **输入**：高压闪电（从 ME 网络提取）+ FE 能量
* **输出**：极高压闪电（存入 ME 网络）
* 需要在设备中安装**闪电坍缩矩阵**
* 每次处理固定产出 1 单位极高压闪电

## 运作流程

1. 选择模式（高压或极高压）
2. 放入所需材料（高压模式需要过载水晶粉，极高压模式需要闪电坍缩矩阵）
3. 接入 FE 能量（最大容量 16,000,000 FE）
4. 特斯拉线圈将自动锁定当前模式并开始充能
5. 充能完成后（约 5 tick），产物会被注入 ME 网络

## GUI 状态说明

| 状态 | 含义 |
|------|------|
| 待机 | 等待输入或模式未锁定 |
| 充能中 | 正在消耗 FE 能量 |
| 等待 FE | FE 能量不足 |
| 等待输入 | 缺少所需材料 |
| 等待 ME 网络 | ME 网络无法接收产出 |
| 准备提交 | 充能完成，即将注入产物 |

## 闪电坍缩矩阵

极高压模式的运行需要在特斯拉线圈的矩阵槽位中安装**闪电坍缩矩阵**。矩阵不会在加工中被消耗，但必须保持安装状态。
