# 闪电系统

闪电是 AE2 闪电科技引入的全新 AE2 存储类型。它像物品和流体一样存在于 ME 网络中，可以被收集、存储和消耗。

## 闪电的种类

闪电分为两个等级：

| 类型 | 描述 |
|------|------|
| **高压闪电** | 基础闪电类型，由闪电收集器在特斯拉线圈放电或受雷击时产生 |
| **极高压闪电** | 高级闪电类型，只有自然雷击才能直接产生，也可由特斯拉线圈从高压闪电升压而来 |

许多配方只需要高压闪电，但部分高级配方需要极高压闪电。

## 闪电的收集

* [闪电收集器](lightning-collector.md) — 捕获雷击，将闪电能量注入 ME 网络
* [特斯拉线圈](tesla-coil.md) — 利用过载水晶粉和 FE 能量批量生产闪电

## 闪电的存储

* [闪电存储元件](lightning-storage.md) — 可放入 ME 驱动器中存储闪电
